# [Lucky Bricks] 확률 고지 안내 (Probability Disclosure)

**Lucky Bricks**를 이용해 주시는 이용자 여러분께 게임 내 적용되는 각종 효과의 등장 확률을 안내해 드립니다.  
We are providing the appearance probability of various effects applied within **Lucky Bricks**.

---

## 1. 콤보 모드 (Combo Mode)
콤보 모드에서는 모든 효과가 동일한 확률로 등장합니다.  
In Combo Mode, all effects appear with the same probability.

| 효과 명칭 (Effect Name) | 상세 기능 (Description) | 등장 확률 (Probability) |
| :--- | :--- | :--- |
| **바 크기 감소 (Bar size down)** | 패들 길이 축소 / Reduces paddle length | **16.67%** |
| **바 크기 증가 (Bar size up)** | 패들 길이 확장 / Increases paddle length | **16.67%** |
| **공 속도 감소 (Ball speed down)** | 공 이동 속도 저하 / Decreases ball speed | **16.67%** |
| **공 속도 증가 (Ball speed up)** | 공 이동 속도 상승 / Increases ball speed | **16.67%** |
| **점수 +100 (Score +100)** | 100점 즉시 합산 / Adds 100 points immediately | **16.67%** |
| **점수 x 2 (Score x 2)** | 현재 점수 2배 / Multiplies current score by 2 | **16.67%** |

> **※ 예외 사항 (Exceptions)**
> * 바 크기가 이미 최소 혹은 최대치에 도달한 경우, 크기가 변하지 않습니다. (The bar size does not change when the bar size is at min or max.)
> * 공 속도가 이미 최소 혹은 최대치에 도달한 경우, 속도가 변하지 않습니다. (The ball speed does not change when the ball speed is at min or max.)

## 2. 로그라이크 모드 (Roguelike Mode)
로그라이크 모드에서는 각 효과의 난이도와 가산점에 따라 확률이 차등 적용됩니다.  
In Roguelike Mode, probabilities are applied differently based on the difficulty and score modifiers.

| 번호 | 효과 명칭 (Effect Name) | 상세 기능 (Detailed Function) | 등장 확률 (Prob.) |
| :---: | :--- | :--- | :---: |
| 1 | **바 크기 감소 (Bar size down)** | 패들 축소 (**브릭당 +3점**) / Paddle Down (**+3 pts per brick**) | **19.23%** |
| 2 | **바 크기 증가 (Bar size up)** | 패들 확장 (**브릭당 -2점**) / Paddle Up (**-2 pts per brick**) | **19.23%** |
| 3 | **공 속도 증가 (Ball speed up)** | 속도 상승 (**브릭당 +3점**) / Speed Up (**+3 pts per brick**) | **19.23%** |
| 4 | **공 속도 감소 (Ball speed down)** | 속도 저하 (**브릭당 -2점**) / Speed Down (**-2 pts per brick**) | **19.23%** |
| 5 | **점수 보너스 (Score Bonus)** | **+100점** 즉시 합산 / **+100 points** added immediately | **19.23%** |
| 6 | **점수 x 2 (Score x 2)** | 현재 점수 **x2배** 적용 / Current score **multiplied by 2** | **3.85%** |
