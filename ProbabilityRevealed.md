# 📊 확률 고지 안내 (Probability Disclosure)

**Lucky Bricks**를 이용해 주시는 이용자 여러분께 게임 내 적용되는 각종 효과의 등장 확률을 안내해 드립니다.  
*We are providing the appearance probability of various effects applied within **Lucky Bricks**.*

---

## 1. 콤보 모드 (Combo Mode)

콤보 모드에서는 모든 효과가 동일한 확률로 등장합니다.  
*In Combo Mode, all effects appear with the same probability.*

| 효과 명칭 (Effect Name) | 등장 확률 (Prob.) |
| :--- | :---: |
| 바 크기 감소 (Bar size down) | **16.67%** |
| 바 크기 증가 (Bar size up) | **16.67%** |
| 공 속도 감소 (Ball speed down) | **16.67%** |
| 공 속도 증가 (Ball speed up) | **16.67%** |
| 점수 +100 (Score +100) | **16.67%** |
| 점수 x 2 (Score x 2) | **16.67%** |

> **※ 예외 사항 (Exceptions)**
> * 바 크기나 공 속도가 이미 최소/최대치에 도달한 경우 수치는 변하지 않습니다.
> * *The bar size or ball speed does not change when it is already at its min or max limit.*

---

## 2. 로그라이크 모드 (Roguelike Mode)

로그라이크 모드에서는 각 효과의 난이도와 가산점에 따라 확률이 차등 적용됩니다.  
*In Roguelike Mode, probabilities are applied differently based on the difficulty and score modifiers.*

| ID | 효과 명칭 (Effect Name) | 등장 확률 (Prob.) |
| :---: | :--- | :---: |
| 1 | 바 크기 감소 (+3 pts per brick) | **19.23%** |
| 2 | 바 크기 증가 (-1 pts per brick) | **19.23%** |
| 3 | 공 속도 증가 (+3 pts per brick) | **19.23%** |
| 4 | 공 속도 감소 (-1 pts per brick) | **19.23%** |
| 5 | 점수 보너스 (+100 points) | **19.23%** |
| 6 | 점수 x 2 (Score x 2) | **3.85%** |
