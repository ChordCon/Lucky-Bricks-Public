# 🎮 럭키 브릭스(Lucky Bricks) 로그라이크 가이드

당신의 선택이 생존과 고득점을 결정하는 전략적 로그라이크 여정이 시작됩니다.

---

## 1. ✨ 핵심 시스템: 리스크와 보상 (Risk & Reward)

유저가 직접 **효과(Modifier)**를 선택하여 게임의 난이도와 점수 효율을 설계합니다. 고득점을 위해 위험을 감수할지, 생존을 위해 안전을 택할지는 여러분의 선택에 달려 있습니다.

---

## 2. 💎 자물쇠 벽돌과 효과(Modifier)

모든 라운드에는 **5개의 자물쇠 벽돌**이 랜덤한 위치에 생성됩니다. 자물쇠 벽돌을 파괴하면 무작위 효과가 등장하며, 라운드 클리어 시 획득한 효과 중 하나를 선택해 누적할 수 있습니다.

### [효과별 획득 확률 및 상세 기능]

| 번호 | 효과 명칭 | 상세 기능 | 등장 확률 |
| :-- | :--- | :--- | :--- |
| 1 | 바 크기 감소 | 패들 길이 축소 (브릭당 +3점) | 19.23% |
| 2 | 바 크기 증가 | 패들 길이 확장 (브릭당 -1점) | 19.23% |
| 3 | 공 속도 증가 | 공 이동 속도 상승 (브릭당 +3점) | 19.23% |
| 4 | 공 속도 감소 | 공 이동 속도 저하 (브릭당 -1점) | 19.23% |
| 5 | 점수 보너스 | 현재 점수 +100점 즉시 합산 | 19.23% |
| 6 | 점수 x 2 | 현재 점수 x2배 적용 | 3.85% |

*   **리롤(Re-roll)**: 라운드당 최대 1회 가능하며, 광고 시청을 통해 목록 중 하나의 효과를 정해진 확률에 따라 교체합니다.
*   **광고 제거 혜택**: '광고 제거' 구매 시, 광고 시청 없이 즉시 리롤이 가능합니다.

---

## 3. ❤️ 라이프(Life) 및 난이도 관리

스테이지가 진행될수록 공의 기본 속도가 점진적으로 빨라지므로 체계적인 라이프 관리가 필수입니다.

*   **최초 라이프**: 총 **3개**(기본 1 + 추가 2)로 시작합니다.
*   **라이프 획득 조건**:
    *   게임 플레이 중 **5콤보 달성 시마다** 라이프 +1 추가
    *   보너스 및 보스 라운드 종료 후 광고 시청 시 추가 라이프 획득 (광고 제거 구매 시 즉시 획득)
*   **보유 제한**: 최대 **5개**까지만 가질 수 있으며, 0이 되면 게임이 종료됩니다.

---

## 4. 👾 보스전 전용 환경 기믹

5 스테이지마다 등장하는 보스전에서는 플레이를 방해하는 특수한 환경 기믹이 발동됩니다.

| 테마 | 기믹 상세 효과 |
| :--- | :--- |
| 🌲 **나무** | 바(Bar) 이동 속도 감소 |
| ⏳ **사막** | 공의 반사(바운드) 각도 무작위성 증가 |
| 🌊 **바다** | 공의 속도가 실시간으로 빨라지거나 느려짐 |
| ⛏️ **광산** | 벽돌의 내구도 증가 (2번 타격 시 파괴) |
| ❄️ **얼음** | 바(Bar)의 마찰력 감소로 인한 미끄러짐 발생 |
| 🔮 **마법** | 공이 불규칙하게 사라졌다가 다시 나타남 |
| 🌋 **화산** | 바(Bar)의 크기가 시간이 지남에 따라 지속 감소 |
| 🌀 **차원** | 균열 진입 시 랜덤한 방향으로 공 재발사 |
| 🔥 **지옥** | **이전 모든 보스 기믹 중첩 적용** + 바(Bar) 좌우 반전 조작 |
